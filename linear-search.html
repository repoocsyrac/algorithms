Linear Search

Key facts

  • Goal: Finds specific element in a list.
  • Best-case time complexity: O(1)
  • Worst-case time complexity: O(N)
  • Average time complexity: O(N)
  • Space complexity: O(1)
  • Pros: Can be used when list is unsorted, can be performed in-place
  • Cons: Poor time complexity
  • When to use: When you are searching a very small list
